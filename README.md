# MiniPlayVerse

이 프로젝트는 유니티2D로 제작되었으며 캐릭터 이동, 미니게임, 점수 시스템 등 간단한 기능을 담고 있습니다.
WASD키로 캐릭터를 움직일 수 있으며 카메라 추적 기능이 있습니다.
특정영역에 진입하면 점수를 볼 수 있고, 미니게임 씬으로 집입할 수 있습니다.
미니게임의 점수가 기록되며 최고 기록이 유지됩니다.
